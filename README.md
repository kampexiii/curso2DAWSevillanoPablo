# Curso 2º DAW - Pablo Sevillano

Repositorio con todas las actividades y prácticas del curso de Desarrollo de Aplicaciones Web (2º año).

## Estructura del Proyecto

### Cliente (JavaScript / HTML / CSS)

Ejercicios y prácticas de desarrollo en el lado del cliente.

- **UT1 - UT5**: Actividades por unidad temática.
- **Exámenes y Correcciones**: Pruebas prácticas y sus resoluciones.

### Servidor (PHP / Laravel)

Desarrollo backend utilizando PHP nativo y el framework Laravel.

#### Ejercicios Destacados (2ª Evaluación)

- **Seguridad en Bases de Datos (Noviembre 28)**:
  - Implementación de consultas seguras para prevenir inyección SQL.
  - Uso de `mysqli` con `bind_param`.
  - Versiones alternativas utilizando `PDO` (PHP Data Objects).
  - Operaciones CRUD (Create, Read, Update, Delete) seguras.
  - Gestión de sesiones y conexiones dinámicas a bases de datos.

### Programación Python

Ejercicios introductorios y avanzados de Python.

- Módulos, funciones y estructuras de control.
- Exámenes y prácticas de lógica.

## Tecnologías

- **Lenguajes**: PHP, JavaScript, Python, HTML5, CSS3.
- **Bases de Datos**: MySQL / MariaDB.
- **Frameworks**: Laravel.
- **Herramientas**: Git, VS Code, XAMPP.
